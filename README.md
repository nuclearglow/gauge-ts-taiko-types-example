A small test project using

* gauge
* gauge-ts
* taiko

# Usage

    $ npm install -g @getgauge/cli
    $ npm i
    $ gauge run specs/

# Initial Setup

    $ npm install -g @getgauge/cli
    $ gauge init ts

Edit `package.json` and manually set `gauge-ts` version to 0.0.5, see this issue here: https://github.com/BugDiver/gauge-ts/issues/57
